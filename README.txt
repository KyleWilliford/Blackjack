A simple text UI game of blackjack.

"Release" EXECUTABLE RUNTIME REQUIREMENTS:
Microsoft Visual C++ 2010 SP1 Redistributable 
(For 64 bit OS - http://www.microsoft.com/en-us/download/details.aspx?id=13523)
(For 32 bit OS - http://www.microsoft.com/en-us/download/details.aspx?id=8328)

"Debug" EXECUTABLE RUNTIME REQUIREMENTS:
An installed version of Visual Studio (2010; maybe 2012, but not tested)

INSTALLATION INSTRUCTIONS
To play the game, simply:
1. navigate to https://github.com/KyleWilliford/Blackjack/blob/master/Blackjack/Release/Blackjack.exe
2. Click "View Raw" in the center of the page.
3. Choose a directory to store the executable.
4. Run the executable (may require a firewall / AV exception)
5. ???
6. Profit.

You could also compile the project in VS or another IDE. 
There may be compatibility errors if you try to compile using GCC or another C compiler (not tested).
